# Linux & DevOps Learning Repository

This repository contains the practical exercises and scripts I write while improving my **Linux system administration, Bash scripting, and DevOps fundamentals**.

The goal of this repository is to document my learning process and build hands-on experience with tools and concepts commonly used in **Linux-based infrastructure and DevOps environments**.

## Topics Covered

* Linux basic commands
* Linux file system
* User and group management
* Basic system administration
* Networking fundamentals
* Linux security basics (firewall, permissions)
* Bash scripting and automation

## Purpose

The purpose of this repository is to:

* Practice Linux commands and system operations
* Develop Bash scripting skills
* Build a strong Linux foundation for DevOps and system engineering
* Track my learning progress with practical examples

All scripts are written as small exercises and experiments while learning.
